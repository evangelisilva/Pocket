# Pocket - AI-Powered Expense Tracker

Pocket is a modern, premium mobile-first web application for tracking your daily expenses. Built with React, Vite, and Firebase, it features a sleek dark-mode glassmorphism design and leverages the power of Google's Gemini AI to automatically extract expense information directly from receipt images.

## Features

- **📸 AI Receipt Scanning**: Snap or upload a picture of your receipt, and Pocket uses Gemini AI to automatically extract the merchant name, date, and total amount.
- **✨ Premium UI/UX**: Designed with a stunning dark theme, neon accents, and smooth glassmorphism effects.
- **📊 Dashboard**: View your total spend for the current month and visualize your daily spending through an interactive chart.
- **💸 Expense Management**: Easily add, edit, or delete manual transactions.
- **📱 PWA Ready**: Installable as a Progressive Web App on iOS and Android devices for a native app-like experience.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind-like custom CSS
- **Backend/Hosting**: Firebase Hosting & Firestore
- **AI Integration**: Google Gemini API
- **Routing**: wouter
- **Icons**: Lucide React
- **Date Handling**: date-fns
